# Medication Tracker

A React + TypeScript web app that allows users to track medications, dosage, and dates. Built with Firebase Authentication and Firestore for persistent cloud data storage.

## Features

- Email/password authentication
- Add medications
- View saved medications
- Delete medications
- User-specific Firestore data

## Tech Stack

- React
- TypeScript
- Firebase Authentication
- Firestore
- Vite

## Run locally

1. Install dependencies:
   npm install

2. Start dev server:
   npm run dev
